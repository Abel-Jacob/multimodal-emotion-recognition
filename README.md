# 🎭 Multimodal Emotion Recognition using Speech and Text

## 📌 Project Overview

This project presents a **Multimodal Emotion Recognition System** developed using the **TESS (Toronto Emotional Speech Set)** dataset. The system combines both acoustic and contextual information to classify emotional states from speech data.

Three independent pipelines were implemented and evaluated:

* 🎙️ Speech Emotion Recognition
* 📝 Text Emotion Recognition
* 🔀 Multimodal Fusion

The project explores temporal acoustic modelling, transformer-based contextual embeddings, and multimodal representation learning.

---

# 🧠 System Architecture

## 🎙️ Speech Emotion Recognition Pipeline

The speech pipeline was designed using a hybrid deep learning architecture consisting of:

* CNN layers for local feature extraction
* BiLSTM layers for temporal sequence modelling
* Attention mechanism for focusing on emotionally relevant speech regions

### Extracted Features

* MFCC
* Delta MFCC
* Mel Spectrogram
* Chroma Features
* Spectral Contrast
* Zero Crossing Rate

The combination of these features enabled the model to learn both spectral and temporal emotional characteristics from speech.

---

## 📝 Text Emotion Recognition Pipeline

The text pipeline used **DistilBERT**, a transformer-based contextual language model.

The textual content associated with the TESS dataset was tokenized and converted into contextual embeddings using DistilBERT, followed by dense classification layers for emotion prediction.

Although transformer models are highly effective for contextual language understanding, the TESS dataset contains highly repetitive and limited textual content, which restricted the effectiveness of the text-only model.

---

## 🔀 Multimodal Fusion Pipeline

The multimodal pipeline combined:

* Speech embeddings from the temporal modelling block
* Contextual embeddings generated by DistilBERT

These representations were fused using feature concatenation and passed through dense classification layers.

The objective of the fusion model was to improve emotional robustness by leveraging both acoustic and contextual modalities simultaneously.

---

# 📂 Dataset

## Toronto Emotional Speech Set (TESS)

The TESS dataset contains emotion-labelled speech recordings collected from female speakers.

### Emotion Classes

* Angry
* Disgust
* Fear
* Happy
* Neutral
* Pleasant Surprise
* Sad

### Dataset Information

* 5600 audio samples
* WAV audio format
* Controlled emotional speech recordings

---

# 📊 Experimental Results

| Pipeline        | Accuracy                       |
| --------------- | ------------------------------ |
| Speech Pipeline | 99.58%                         |
| Text Pipeline   | 16.67%                         |
| Fusion Pipeline | Improved multimodal robustness |

The speech pipeline achieved the strongest standalone performance, indicating that acoustic features contained highly discriminative emotional information.

The text pipeline achieved significantly lower accuracy due to the limited diversity of textual content in the TESS dataset. Since many samples share similar textual structures, contextual embeddings alone were insufficient for reliable emotion separation.

The multimodal fusion approach improved robustness by combining contextual and acoustic information, reducing ambiguity in emotionally similar classes.

# 📝 Text Pipeline Analysis

## Training Curves

![Text Training Curves](results/plots/text_training_curves.png)

The text training curves indicate weaker learning performance compared to the speech model.

The model struggled to generalize effectively due to the repetitive and limited textual structure of the TESS dataset.

Unlike natural conversational datasets, TESS provides minimal contextual diversity, limiting DistilBERT’s ability to learn discriminative emotional semantics.

---

## Text Confusion Matrix

![Text Confusion Matrix](results/plots/text_confusion_matrix.png)

The confusion matrix reveals significant overlap between emotional categories.

### Key Observations

* Strong bias toward dominant prediction classes
* Poor separation between emotionally similar labels
* Reduced contextual distinguishability

These results confirm that textual content alone was insufficient for robust emotion recognition in this dataset.

---

## Text ROC Curve

![Text ROC Curve](results/plots/text_roc_curve.png)

The ROC curves demonstrate weaker discrimination compared to the speech model.

Lower AUC values indicate reduced class separability and limited contextual information available within the dataset text.

Despite weaker standalone performance, DistilBERT embeddings still contributed complementary contextual information useful for multimodal fusion.

# 🔍 Error Analysis

Several challenging emotional pairs resulted in misclassification:

| Emotion Pair                | Reason                                     |
| --------------------------- | ------------------------------------------ |
| Fear ↔ Sad                  | Similar acoustic energy and pitch patterns |
| Neutral ↔ Pleasant Surprise | Subtle emotional intensity differences     |
| Sad ↔ Neutral               | Low spectral variation                     |

These errors primarily occurred in lower-intensity emotional expressions with overlapping temporal characteristics.
